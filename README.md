# Netflix-Insights
## **Project Topic: Netflix Movies & TV Shows Insight**

The Netflix Movies & TV Shows Insights project analyzes data from the Netflix catalog to uncover patterns, trends, and insights about the platform’s content library.
The dataset includes key information such as title, type (movie or TV show), release year, country, genre, cast, director, and date added to Netflix.

By applying Exploratory Data Analysis (EDA) and data visualization, the project explores:  
Content distribution across countries and years  
Popular genres and trends over time  
Comparison between movies and TV shows  
Insights into directors, actors, and regional content diversity  

This analysis helps understand Netflix’s global strategy, evolving content focus, and audience preferences.

**CSV File:**  
•	Name: netflix_titles.csv

**Source URL:**  
• Kaggle Netflix Dataset
•	https://www.kaggle.com/datasets/shivamb/netflix-shows

**Number of Rows and Columns:**  
•	Rows: 8807  
•	Columns: 12 (Title, Director, Cast, Country, Date Added, Release Year, Rating, Duration, Genre, Description, Type)

**Aim of Your Project:**  
To analyze and visualize patterns within the Netflix dataset to uncover meaningful insights about content types, countries, genres, and release trends over time.
